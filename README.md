# Cybersecurity Incident Response Notes

A simple student-friendly cybersecurity mini project covering common incident response scenarios, basic security review steps, and short notes on important figures in tech and cybersecurity.

## Contents

- `answers.md` — concise answers to common cybersecurity incident-response questions
- `security_checklist.md` — short application security checklist
- `incident_response_template.md` — basic incident response workflow template

## Topics Covered

- Phishing email triage
- Malware infection response
- Ransomware containment
- Unauthorized access attempts
- Data breach suspicion handling
- Web application vulnerability review
- Network compromise investigation
- Risk management and AI security
- Security checklist basics
- Cybersecurity personalities

## Purpose

This repo is intended as a simple educational project for demonstrating cybersecurity understanding in a clear and practical format.
